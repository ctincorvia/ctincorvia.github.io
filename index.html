Week 1

by Charles Incorvia

Posted on January 23, 2017

The first of many blog posts!

What did you do this past week?
So far, we've discussed some handy technology tools for developing in groups of people (Github, TravisVI, Docker) and we've learned about what ++ and += do in different programming languages and contexts. The reading that's been assigned so far hasn't been bad. Some stuff about how to succeed in college (A little late for me, sadly), some handy tips about being a programmer in the real world, and a whole lot about the overarching design paradigm of object oriented programming.

What's in the way?
So far there's hasn't been a whole lot. I imagine we'll move on to the first project and it'll be a hurdle. Apparently, the coding itself is not difficult, but integrating the technology tools together is the hardest part. I'm not looking forward to that. I've never worked with HTML or CSS before, so getting this thing up has been the most difficult bit so far.

What will you do next week?
The first project will be assigned next week. I hope to get it knocked out relatively early, contrary to my programming habits before this point. You'd think after 3.5 years I'd learn. Keeping up with the reading is vital for quiz performance in this class, which is good for me, because otherwise I'll just do it all the week before the test and have a horrible time. So I'll *hopefully* be doing lots of reading this week.

What's the tip of the week?
Go find a template online to help with your blog post. There's quite a few out there for free. I would have a blank background with just white text otherwise. Or if you're some CSS wizard you could do it yourself.
