Week 13

by Charles Incorvia

Posted on April 10, 2017

The Tenth Blog Post!

What did you do this past week?
This week I spent finishing the life project. Because I had nearly all of the coding done, this week was spent hunting for errors from unexpected memory leaks, double frees, and unexpectedly incorrect input. Thankfully, the project was turned in well before the deadline with all parts completed and all tests passed, in in contrast to the last lab.

What will you do next week?
The coming week will be spent studying. I have some ground to make up from the last exam, with a decidedly below-average score, so if I want a good grade I have to perform.

What's in the way?
I have quite a few things I have to wrap up before the semester ends. I've got housing, I've got a job offer I need to consider, and I've got to find a subleaser and a person to fill my slot beyond the lease. There is also a final project looming in another class. The only things in the way of my success in this class is my other commitments at the moment.

What's my experience in the class?
The class is as always fair. But I personally am wonder if this class is more about C++ rather than object oriented programming in general. Though we certainly learn object oriented principles, and C++ is an object oriented language, lot of this class is about minute details and operational semantics in C++, which is not what I expected when I began the class.

What's the tip of the week?
Plan for duplicate inputs in the lab! I had the hardest time figuring out what was wrong with my hackerrank tests until I discovered I wasn't handling duplicate input, though I should have been.
